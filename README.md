# Disney Movies Data Analysis

## Overview
This project analyzes a Disney movies dataset using Microsoft Excel. The dataset was cleaned and transformed from a messy raw format into a structured dataset for analysis.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Excel formulas

## Data Cleaning Performed
- Cleaned movie names
- Standardized text columns
- Extracted release year
- Cleaned budget and box office values
- Removed unwanted characters and inconsistencies
- Handled missing and messy values

## Analysis Performed
- Calculated profit using Box Office - Budget
- Counted loss-making movies
- Created pivot tables for:
  - Release Year vs Box Office
  - Country vs Profit
- Created charts to visualize trends

## Key Insights
- Movies released in later years generated higher total box office revenue
- United States contributed the highest total profit in the dataset
- Higher budget did not always guarantee higher profit
- 40 movies in the dataset resulted in a loss
